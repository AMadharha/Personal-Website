Introduction

During this work term, I worked as a Data Analyst Co-op at RBC, one of Canada's largest banks. This opportunity helped me understand how data is used in real-world financial services and gave me a chance to apply what I've learned in school to real projects. I worked on tasks related to wire transfers, employee attrition, Tableau reporting, and ETL (Extract, Transform, Load) processes. I also had the chance to improve my technical skills in Python, SQL, and tools like Selenium. This report summarizes the work I did, what I learned, and how the experience helped me grow as a student and future data professional.

Company Overview

RBC is one of the biggest financial institutions in Canada. It provides banking, investment, insurance, and wealth management services. RBC uses a lot of data to make decisions, support clients, and improve operations. As a Data Analyst Co-op, I worked in a team that handled various datasets and worked with tools to clean, process, and visualize data.

Role and Responsibilities

My role as a Data Analyst Co-op involved the following tasks:

ETL (Extract, Transform, Load) Work

One of my main responsibilities was creating and managing ETL pipelines using Python and SQL Server. I worked with datasets related to wire transfers and employee attrition. I learned how to:

  • Connect to databases
  • Extract raw data
  • Clean and transform data into usable formats
  • Load the data into SQL Server tables

These ETL pipelines were important for creating dashboards and reports used by different teams at RBC.

Automation with Python and Selenium

I also helped with automation tasks using Python. One project involved using Selenium, a browser automation tool, to automatically update reports and process records that would normally take a lot of time manually. This helped our team and other departments save time and reduce errors.

Working with Tableau

I worked on Tableau dashboards by analyzing metadata like workbook names, view names, and click counts. I used this information to measure dashboard usage and help teams understand which views were most valuable. This work helped make our dashboards more useful and easier to manage.

Data Lineage Project

I helped with a data lineage project, which involved mapping out relationships between SQL tables, Tableau dashboards, Python scripts, and SAS files. The goal was to show how data flows across systems. I used Python to build data structures and explored ways to visualize these connections using D3.js, a JavaScript library for data visualization.

Collaboration and Communication

I worked closely with my manager and other team members. We had regular meetings to go over progress, plan new work, and share updates. I also had 1-on-1 meetings with my manager to get feedback and talk about my goals.

Tools and Technologies Used

During the term, I got hands-on experience with many different tools:

  • Python - for scripting, automation, data processing
  • SQL Server - for managing and querying structured data
  • Selenium - for automating web tasks and repetitive updates
  • Tableau - for data visualization and dashboard analysis
  • D3.js - for interactive data visualizations
  • Git/GitHub - for version control and collaboration
  • Excel - for quick data checks and sharing with other teams
  • Teradata, Hive/Hadoop - for retrieving data from RBC's enterprise data systems

These tools helped me work on different kinds of data projects and improved my technical confidence.

Challenges and Solutions

One challenge I faced was working with unfamiliar datasets and systems. For example, learning how RBC stores data in their Enterprise Data Warehouse (EDW) and Enterprise Data Lake (EDL) was new for me. I overcame this by asking questions, reading documentation, and breaking down tasks into small steps.

Another challenge was automating processes that involved complex web pages. Using Selenium helped, but it took trial and error to get it working reliably. I learned how to use debugging tools and write better code through this experience.

Conclusion

My co-op term at RBC was a great experience. I learned a lot, gained practical skills, and got to work with amazing people. The projects I worked on helped me understand how data plays a key role in banking, and the feedback I received helped me grow.

This experience has confirmed that I want to keep learning more about data science and analytics. I'm excited to apply what I've learned in future courses and jobs.
